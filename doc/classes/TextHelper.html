truncate_html(text, *args)

Returns a given text truncated after a given length if text is longer than length (defaults to 30). The last characters will be replaced with ommission.

It‘s just like the Rails truncate helper but doesn‘t break HTML tags, entities, and optionally, words.

Options

  • length - Maximum length of truncated text, including the length of ommission (default: 30)
  • ommission - Ellipsis, only present if text has actually been truncated (default: ’…’).
  • break_on_word - true to ensure that text is truncated on a word boundary. Note this may mean that the length of the truncated text returned is less than length (default false)
  • coda - String that will be appended to text after truncation regardless of whether present regardless text is truncated or not (default: nil)

link clean punctuation

Examples

  truncate '<p>Hello, my name is Danbert</p>'
  #=> '<p>Hello, my name is Danbert</p>'

  truncate '<p>Hello, my name is Danbert</p>', :length => 15
  #=> '<p>Hello, my na...</p>'

  truncate '<p>Hello, my name is Danbert</p>', :length => 15, :break_on_word => true
  #=> '<p>Hello, my...</p>'

  truncate '<p>Hello, my name is Danbert</p>', :coda => link_to('More', @user)
  #=> '<p>Hello, my name is Danbert</p><a href="/users/1">More</a>'

  truncate '<p>Hello, my name is Danbert</p>', :length => 15, :coda => link_to('More', @user)
  #=> '<p>Hello, my na...</p><a href="/users/1">More</a>'
